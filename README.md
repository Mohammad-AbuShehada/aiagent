# AI Agent

A command-line AI coding agent built with Python and the Google Gemini API.

Developed during the **Boot.dev Stage** of the **Foothill Technology Solutions Internship Program**.

## Features

- Function calling
- File and directory management
- File reading and writing
- Python code execution
- Multi-step agent loop
- Secure working directory

## Tech Stack

- Python
- Google Gemini API
- google-genai
- python-dotenv
- uv

## Run

```bash
uv run main.py "Your prompt here"
```

## Environment

Create a `.env` file:

```env
GEMINI_API_KEY=your_api_key
```
